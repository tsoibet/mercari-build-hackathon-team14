## :beginner: How to use

1. Clone this repository and change directory to this project

2. Run the commands below in a terminal:

```
cd backend
uvicorn main:app --port 9000
```

3. run the commands below in an other terminal:

```
cd frontend
npm ci
npm start
```

3. Visit http://localhost:3000
